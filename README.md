# 🧠 MoodCast

**An AI-powered mood-based daily planner with personalized suggestions, food ordering, journaling, and wellness features.**

## ✨ Features

- **AI-Powered Day Plans** — Tell MoodCast how you're feeling and get a personalized daily plan generated by Google's Gemini AI
- **Mood-Based Food Suggestions** — Get food recommendations tailored to your emotional state with direct links to Uber Eats and PickMe Food
- **Daily Journal** — Write notes about your day and save them with your mood entry
- **Mood History Tracker** — View your mood patterns over the past week with an interactive emoji chart
- **Spotify Playlists** — Pre-curated playlists for each mood to set the perfect vibe
- **Breathing Exercises** — Guided breathing techniques matched to your current emotional state
- **Daily Affirmations** — Personalized motivational messages for each mood
- **Streak Counter** — Track how many days in a row you've checked in
- **Dark Mode** — Easy on the eyes with a beautiful dark theme toggle
- **100% Client-Side** — All data stored locally using localStorage, no backend needed

## 🎯 Moods Supported

😊 Happy | 😰 Anxious | 😴 Tired | 🎯 Focused | 😢 Sad | 🚀 Excited

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- A Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Adithya-Ekanayake/moodcast.git
cd moodcast
```

2. **Install dependencies**
```bash
npm install
```

3. **Add your Gemini API key**

Open `src/App.js` and find this line:
```js
const GEMINI_KEY = "YOUR_ACTUAL_KEY_HERE";
```

Replace `YOUR_ACTUAL_KEY_HERE` with your actual Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey).

4. **Start the development server**
```bash
npm start
```

The app will open at `http://localhost:3000`

## 📱 How to Use

1. **Select Your Mood** — Click on one of the 6 mood buttons at the top
2. **Generate AI Plan** — Click "✨ Generate AI Day Plan" to get personalized suggestions
3. **Explore Features** — Switch between tabs:
   - **📋 Plan** — Your personalized daily schedule
   - **🍽️ Food** — AI-suggested foods with ordering links
   - **📝 Journal** — Write your thoughts for the day
   - **🧘 Wellness** — Spotify playlist, breathing exercise, affirmation
4. **Save Your Check-In** — Click "📌 Save today's mood & journal" to record your entry
5. **Track Progress** — View your mood history and maintain your streak

## 🎨 Features Breakdown

### AI Day Plan Generation
Uses Google Gemini AI to generate unique, personalized daily schedules based on your mood:
- Morning routine suggestions
- Afternoon activities
- Evening wind-down ideas
- Motivational quotes

### Smart Food Recommendations
AI suggests 3 foods/drinks that match your emotional state with:
- Why it fits your mood
- Direct links to order via Uber Eats
- Local delivery option via PickMe Food (Sri Lanka)

### Mood Tracking
- Visual weekly chart showing your mood patterns
- Emoji representation for quick recognition
- Stored mood entries with timestamps and journal notes
- Accessible mood history for past entries

### Wellness Features
- **Breathing Exercises** — Scientifically-backed techniques (box breathing, 4-7-8 technique, etc.)
- **Curated Spotify Playlists** — Pre-selected playlists for each mood
- **Daily Affirmations** — Mood-specific motivational messages
- **Streak Counter** — Gamification to encourage daily check-ins

## 🔐 API Keys

This app requires a free Gemini API key:

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy your key and add it to `src/App.js`

**Free tier includes:**
- Up to 60 requests per minute
- Sufficient for personal use

## 💾 Data Storage

All data is stored locally in your browser using localStorage:
- Mood history (last 30 entries)
- Daily streak count
- Journal entries
- Theme preference (light/dark mode)

**No data is sent to any server** — your information stays completely private!
